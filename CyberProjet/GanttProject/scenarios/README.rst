Ce répertoire contient des exemples de scénarios d'utilisation de GanttProject_.
Ces scénarios permette de valider le modèle ``GanttProject.use``.
Les scenarios sont écrits sous forme de fichier ``.soil`` modélisant le contenu
des fichiers ``.gan`` présent dans le répertoire ``resources``.
